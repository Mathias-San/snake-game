Snake Game 🐍

A classic Snake game built from scratch with Python and Pygame. This is a portfolio project demonstrating core game development concepts including game loops, collision detection, sprite movement, and score tracking.


Demo


Control the snake, eat the food, grow longer — don't hit the walls or yourself!




Features


Smooth grid-based snake movement
Random food spawning that never overlaps the snake
Score tracking with a persistent high score
Progressive difficulty — the snake speeds up every 5 points
Self-collision and wall-collision detection
Game over screen with score summary
Press R to instantly restart
Reverse-direction prevention (can't run into yourself by accident)



Built With


Python 3.12
Pygame



Getting Started

Prerequisites

Make sure you have Python and Pygame installed.
